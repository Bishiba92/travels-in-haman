About:



This game is being developed as part of a project for a school course. The gameplay time is expected to be within 10-30 minutes.


Movement Controls:
Move left: Left arrow, A-key.
Move up: Up arrow, W-key.
Move right: Right arrow, D-key.
Move down: Down arrow, S-key.

Player Screen:
Equip item: Left-click inventory item.
Un-Equip item: Left-click equipped item.
Destroy item: Right-click inventory item.
Use Potion: Left-click potion in inventory.

Add Inventory:
Click the button "Add random item" in order to add an item to the inventory.
