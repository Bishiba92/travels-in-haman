Planned Features:


Overall we are going to be adding more features into the game that makes gameplay interesting, at the moment it just a framework. The goal is to have a combat system that moves the game forwards. We want to add skills and also add more stats than the current three stats.
Hopefully the features below will be added during the coming five weeks.

Features overview
  1. Skills
  2. Levelsystem
  3. Stats
  4. Combat
  5. World
  6. Client-side saving
  7. Animated character movement
Skills
The player gets to pick between a tree of fire, ice and healing skills.
Skills are granted upon levelling up.
  1. Five fire skills
  2. Five ice skills
  3. Three healing skills

Levelsystem
The player will get experience points for completing quests and winning fights.
The listed additions are to be implemented.
  1. Experience points
  2. Experience bar
  3. Bonuses for leveling up, stats and skills.

Combat
As in any game there will be a challenge, in this game it will be presented in the
shape of bandits, orcs and wolfs. The combat system will feature the below additions.
  1. Artificial intelligence(Target picking and healing/attacking/defending)
  2. Turn-based combat
  3. Level appropriate challenges

Stats
  1. Magic attack and defense
  2. Critical hit chance
  3. Luck - affecting drop and gold rates
  4. And more

World
  1. Scrollable and bigger map
  2. Interacting with map-objects
  3. More towns
  4. Story line
